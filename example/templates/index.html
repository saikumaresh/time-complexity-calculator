What is Big O Notation?

Big O notation is a mathematical representation used to describe the efficiency of algorithms in terms of their time complexity. It provides a way to express how an algorithm's runtime scales with the size of the input data. Here are some common complexities:

  • O(1): Constant time, regardless of input size.
  • O(n): Linear time, scales proportionally with input size.
  • O(n log n): Logarithmic time, common in efficient sorting algorithms.
  • O(n^2): Quadratic time, typically occurs with nested loops.
  • O(2^n): Exponential time, seen in recursive algorithms with many branches.
  • O(n!): Factorial time, very inefficient, found in brute force solutions.
